# Daily Mood vs Productivity Predictor

## 📌 Project Overview
This project predicts a person's productivity level based on their daily habits such as sleep, study hours, screen time, and mood.

The goal of this project is to demonstrate how basic machine learning techniques can be applied to solve real-life problems in a simple and effective way.

---

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn

---

## 📊 Dataset Description
The dataset used in this project is manually created and contains the following features:

- Sleep: Number of hours slept  
- Study: Number of hours spent studying  
- Screen: Screen time in hours  
- Mood: Mood level (0 = low, 1 = medium, 2 = high)  
- Productivity: Output level (0 = low, 1 = medium, 2 = high)  

Each row represents a real-life scenario of daily habits and corresponding productivity.

---

## 🧠 Model Used
A Decision Tree Classifier is used to train the model. The dataset is split into training and testing sets to evaluate performance.

---

## 💡 Key Insight
The project shows that balanced habits such as proper sleep, moderate study time, limited screen usage, and a positive mood can lead to higher productivity.
